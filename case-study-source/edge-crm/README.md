# Edge CRM — Case Study Source (from Figma)

Source material for the **Edge CRM** UX case study, pulled directly from the Figma
canvas on 2026-07-06. This is the raw, structured record of the design story so it
can be written up on the portfolio site (`content/case-studies.ts` →
`/case-studies/edge-crm`).

**Product:** Edge CRM — an AI-driven B2B platform where sales & support reps at
**12 Grids** log, track, and resolve client cases, often mid-call.

**What's here vs. what's already on the site:**

- The site's `edge-crm` entry currently documents the **Add Case** screen in depth
  (9 acts, Hero's Journey structure).
- This folder additionally captures the **Case List** redesign chapter — the story
  teased as "Next" on the site but not yet written up: figure-ground, F-shaped
  reading pattern, before/after, and the card component evolution.
